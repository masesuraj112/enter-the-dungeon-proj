# Enter the Dungeon (2D Java Game)

## Overview
This is a 2D platform game inspired by Enter the Gungeon, built in Java.  
The main goal of the project was to practice object-oriented programming and build a structured game system from scratch.

I focused on designing the game using OOP principles and making the code modular so different parts of the game (player, enemies, mechanics) are separated properly.

---

## What the game includes
- Basic 2D movement and platform-style gameplay
- Player and enemy entities
- Simple combat / attack system
- Collision detection
- A game loop that updates everything each frame

---

## Object-Oriented Design
The project was built mainly around OOP concepts:

- **Encapsulation** – each entity handles its own state like position and health  
- **Inheritance** – shared behaviour between game objects  
- **Polymorphism** – different behaviours for different entities (especially attacks and updates)  
- **Abstraction** – common structure for game objects

The idea was to keep things flexible so new entities or mechanics can be added without rewriting a lot of code.
